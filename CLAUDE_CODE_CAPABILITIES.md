# Claude Code 能力介绍

## 概述

Claude Code 是 Anthropic 推出的基于命令行的 AI 编程助手，深度集成在终端和 IDE 中，能够帮助开发者完成各类软件工程任务。

## 核心能力

### 1. 代码理解与搜索
- 基于正则表达式和 glob 模式的高效代码搜索
- 跨文件代码探索与符号查找
- 代码结构分析和依赖关系梳理

### 2. 代码编辑
- 精确的字符串匹配替换编辑
- 支持单文件和多文件批量修改
- 保持代码风格一致性

### 3. 项目管理
- 文件创建、读写和管理
- Git 工作流支持（提交、分支管理、PR 创建）
- 工作树（Worktree）隔离开发支持

### 4. 任务规划与执行
- 复杂任务自动拆解为子任务并跟踪进度
- 支持 Plan Mode 进行实现方案设计
- 多代理并行工作，提高效率

### 5. Web 能力
- 网页内容获取与解析
- 网络搜索获取最新技术信息

### 6. 自动化与维护
- 定时任务调度（Cron 支持）
- 持续循环执行（/loop）
- 会话记忆系统，记录用户偏好和项目上下文

### 7. IDE 集成
- VS Code 原生扩展支持
- JetBrains 系列 IDE 支持
- 终端 APP 和 Web App 支持

### 8. 安全与权限
- 分层权限管理（用户级、项目级）
- 命令沙箱执行
- 危险操作确认机制

### 9. 技能扩展
- 安全代码审查
- API 构建与调试
- PR Review
- 配置管理

### 10. 上下文管理
- 自动消息压缩，支持长时间对话
- 持久化记忆，跨会话保留关键信息
- 工作目录感知与项目管理

## 使用方式

- **终端 CLI**：`claude` 命令直接在终端启动
- **IDE 插件**：在 VS Code / JetBrains 中直接使用
- **Web App**：通过浏览器访问 claude.ai/code

## 获取帮助

在会话中输入 `/help` 可获取完整的使用指南和相关命令参考。
